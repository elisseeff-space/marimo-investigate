Проанализируй документ и извлеки информацию о национальных целях развития, 
национальных проектах, федеральных проектах, государственных программах и их показателях.

## Инструкция:

1. Внимательно прочитай документ
2. Идентифицируй все национальные цели развития (НЦР)
3. Для каждой НЦР найди связанные показатели и индикаторы
4. Идентифицируй национальные проекты (НП) и федеральные проекты (ФП)
5. Извлеки все числовые показатели с целевыми значениями по годам
6. Определи связи между элементами
7. Выведи результат в формате JSON согласно JSON Schema
